PROJECT IDEA

MY PROFILE INTEREST IN IT IDEAL JOB PROJECT IDEA
Overview
My Project idea is to manufacture and program a small VPN add-on module that can be added in line to any modem/router for under $100. Currently most modem Routers do not come with a built in VPN feature. This idea would bring better security to the masses and would help prevent website tracking. This would also make it more affordable than modem/router replacement. Currently there is no affordable easy to install addon modules of this type. I see it is a product that can be sold world wide and would help most people to improve there security in some form or way.

Motivation
My Motivation and the reason for my project idea stems from the lack of hardware based VPN modules that can be easily be added to an existing modem/router to protect the whole house. Yes there is software based programs, but they do not protect the complete house and need to be installed on each device. Most modern day routers, especially ones provided by internet providers usually don't have a built in VPN feature. My motivation also stems from the lack of privacy and the greater need to improve security, to enable tracking provision, greater freedom and the end to geo-location blocking and control. After looking for this type of module, i found that there was no module of this type on the market. With trillions of internet connections all over the world there is a lot of people who are at greater risk.


Description
The VPN add-on module will provide a way to give freedom and protection back to the the user. It will be compatible with VPN connection types such as L2TP/IPsec, PPTP, SSTP, OPENVPN and will be allow easy connection and setup to VPN services such as ExpressVPN, NordVPN, CyberGhost, IPVanish, Surfshark, PrivateVPN, PureVPN and many more. The Module will allow better security, better Online anonymity and will provide a way to unblock websites and bypass filtering. it will allow the user to hide there actual Ip address and can help overcome bandwidth throttling used by some internet providers.This module will help shield your activity from Hackers and malware. It will provide a way for devices that are not able to add vpn software, and fall outside of the Android, Apple OSX, Windows or linux curtain to have some protection.

Tools and Technologies
The tools that would be needed to be able to complete this project would be Autodesk Circuit Design Software which is used for electronic design and simulation, Eagle PCB Design Software for the design and layout of the circuit board. I would also need to find and work with a good reliable circuit board manufacturer to ensure quality and reliability and possibly microprocessor programming. On the Software side we could use an open source software like atom and GitHub for source control. If programming the microprocessor in house we would also need to include a microprocessor programmer from a good reliable manufacturer. we would also need to raise funds for a startup. which we could possibly put online so there would be some web design involved.

Skills Required

The skills required for a project like this would be, electronic design and manufacturing knowledge, C++ Programming skills, communication and marketing skills. It would involve research and development around connection types such as L2TP/IPsec, PPTP, SSTP OpenVPN, research and development of services such as ExpressVPN etc.

Outcome

Upon successful application and by keeping within design and cost targets, we expect to be able to market and sell millions of units world wide. This might not instantly protect all users as there is trillions of users around the world, but will gives users around the world a simple cost effective way to better protect themselves online. This will provide a solution that does not already exist, and will help protect many millions of people world wide.
